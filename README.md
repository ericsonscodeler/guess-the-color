## Getting Started


Migrate prisma para salvar os score.

```bash

npx prisma migrate dev

```

Iniciar aplicação

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Visualizar dados no prisma studio.

```bash

npx prisma studio

```

Rodar os testes unitários

```bash

npm run test

```
